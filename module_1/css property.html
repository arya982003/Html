WEB APPLICATION DEVELOPMENT

Web application development is the creation of application programs that reside on remote servers and are delivered to the user's device over the Internet.
It should be performed while applying three major operations.

  1. HTML
  2. CSS
  3. JAVASCRIPT

HTML

HTML stands for HyperText Markup Language, is the code that is used to structure a web page and its content.

CSS

CSS stands for Cascading Style Sheets, and it's used to add style to web page by dictating how a site is displayed on a browser. it's a language used to style HTML elements.

JAVASCRIPT

Javascript is used by programmers across the world to create dynamic and interactive web content like applications and browsers.
